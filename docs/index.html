Project Overview

This repository contains a tiny Express API plus a GitHub Pages site that explains the CI/CD pipeline. Every push triggers automated tests, packaging, and publishing of the documentation site. The demo is personalized for Cloud Computing with Dr. Ahmad.

  • Stack: Node.js, Express, Jest, GitHub Actions
  • Deployment targets: Docker-ready API + GitHub Pages
  • Goal: show full lifecycle from commit to deployment

CI/CD Pipeline

  1. CI: Run lint + unit tests on every push/PR.
  2. Package: Produce a downloadable ZIP artifact for the examiner.
  3. CD: Publish this page automatically via GitHub Pages.

How to Reproduce

git clone https://github.com/NPaugust/cloud-ci-showcase.git
cd cloud-ci-showcase
npm install
npm run dev

Update the workflow files with secrets (if pushing to Docker Hub or other cloud) and re-run the pipelines to see automation in action.
